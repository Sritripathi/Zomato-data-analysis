# Zomato-data-analysis
This is a Data Analytics project where we have analyzed the data of zomato and found out the key insights using python and its libraries. 
 The steps involved in analysing the zomato data is:
 1. Get the data set from online sources like kaggle.
 2. Open jupyter notebook or google colab, import important libraries like pandas,numpy,matplotlib,seaborn etc.
 3. Load the dataset
 4. Data cleaning and pre-processing
 5. Exploratory data analysis
 6. Visual representation of key insights and finding
 7. Find out key insights and present it.
